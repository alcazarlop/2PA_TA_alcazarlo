A wrapper for sqlite3_exec() function.

The function calls the sqlite3_exec() with the desired callback, it checks that the query and the callback are properly setted but not the data_t. It does not return error when the query or the callback fails

Parameters
queryThe query that will be executed by the callback
data_tThe data where the information retrieve for the callback will be stored
sqlite3_callbackThe sqlite3 callback that will be executed
