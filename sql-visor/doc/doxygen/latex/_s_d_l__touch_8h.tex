Get the number of registered touch devices.

On some platforms S\+DL first sees the touch device if it was actually used. Therefore \doxyref{S\+D\+L\+\_\+\+Get\+Num\+Touch\+Devices()}{p.}{_s_d_l__touch_8h_a14a41a2964ddf868a180e42ccfd441d1} may return 0 although devices are available. After using all devices at least once the number will be correct.

This was fixed for Android in S\+DL 2.\+0.\+1.

the number of registered touch devices.
