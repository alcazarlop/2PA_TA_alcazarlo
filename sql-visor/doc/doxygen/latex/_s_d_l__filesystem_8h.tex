Get the user-\/and-\/app-\/specific path where files can be written.

Get the \char`\"{}pref dir\char`\"{}. This is meant to be where users can write personal files (preferences and save games, etc) that are specific to your application. This directory is unique per user, per application.

This function will decide the appropriate location in the native filesystem, create the directory if necessary, and return a string of the absolute path to the directory in U\+T\+F-\/8 encoding.

On Windows, the string might look like\+:

{\ttfamily C\+:\textbackslash{}\textbackslash{}Users\textbackslash{}\textbackslash{}bob\textbackslash{}\textbackslash{}App\+Data\textbackslash{}\textbackslash{}Roaming\textbackslash{}\textbackslash{}My Company\textbackslash{}\textbackslash{}My Program Name\textbackslash{}\textbackslash{}}

On Linux, the string might look like"

{\ttfamily /home/bob/.local/share/\+My Program Name/}

On Mac OS X, the string might look like\+:

{\ttfamily /\+Users/bob/\+Library/\+Application Support/\+My Program Name/}

You should assume the path returned by this function is the only safe place to write files (and that \doxyref{S\+D\+L\+\_\+\+Get\+Base\+Path()}{p.}{_s_d_l__filesystem_8h_a752d42a2e7fbff1292b7e54593be37bc}, while it might be writable, or even the parent of the returned path, isn\textquotesingle{}t where you should be writing things).

Both the org and app strings may become part of a directory name, so please follow these rules\+:


\begin{DoxyItemize}
\item Try to use the same org string ({\itshape including case-\/sensitivity}) for all your applications that use this function.
\item Always use a unique app string for each one, and make sure it never changes for an app once you\textquotesingle{}ve decided on it.
\item Unicode characters are legal, as long as it\textquotesingle{}s U\+T\+F-\/8 encoded, but...
\item ...only use letters, numbers, and spaces. Avoid punctuation like \char`\"{}\+Game
  Name 2\+: Bad Guy\textquotesingle{}s Revenge!\char`\"{} ... \char`\"{}\+Game Name 2\char`\"{} is sufficient.
\end{DoxyItemize}

The returned path is guaranteed to end with a path separator (\textquotesingle{}\textbackslash{}\textquotesingle{} on Windows, \textquotesingle{}/\textquotesingle{} on most other platforms).

The pointer returned is owned by the caller. Please call S\+D\+L\+\_\+free() on the pointer when done with it.
