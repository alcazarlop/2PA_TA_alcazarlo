{\footnotesize\ttfamily const D\+E\+C\+L\+S\+P\+EC char$\ast$ S\+D\+L\+C\+A\+LL S\+D\+L\+\_\+\+Get\+Error (\begin{DoxyParamCaption}\item[{void}]{ }\end{DoxyParamCaption})}

Retrieve a message about the last error that occurred on the current thread.

It is possible for multiple errors to occur before calling \doxyref{S\+D\+L\+\_\+\+Get\+Error()}{p.}{_s_d_l__error_8h_af3cc615dad0d5ed6b36cb84cab15ecb6}. Only the last error is returned.

The message is only applicable when an S\+DL function has signaled an error. You must check the return values of S\+DL function calls to determine when to appropriately call \doxyref{S\+D\+L\+\_\+\+Get\+Error()}{p.}{_s_d_l__error_8h_af3cc615dad0d5ed6b36cb84cab15ecb6}. You should {\itshape not} use the results of \doxyref{S\+D\+L\+\_\+\+Get\+Error()}{p.}{_s_d_l__error_8h_af3cc615dad0d5ed6b36cb84cab15ecb6} to decide if an error has occurred! Sometimes S\+DL will set an error string even when reporting success.

S\+DL will {\itshape not} clear the error string for successful A\+PI calls. You {\itshape must} check return values for failure cases before you can assume the error string applies.

Error strings are set per-\/thread, so an error set in a different thread will not interfere with the current thread\textquotesingle{}s operation.

The returned string is internally allocated and must not be freed by the application.

\begin{DoxyReturn}{Returns}
a message with information about the specific error that occurred, or an empty string if there hasn\textquotesingle{}t been an error message set since the last call to \doxyref{S\+D\+L\+\_\+\+Clear\+Error()}{p.}{_s_d_l__error_8h_a007ed517953a99a46e46be567ef5609e}. The message is only applicable when an S\+DL function has signaled an error. You must check the return values of S\+DL function calls to determine when to appropriately call \doxyref{S\+D\+L\+\_\+\+Get\+Error()}{p.}{_s_d_l__error_8h_af3cc615dad0d5ed6b36cb84cab15ecb6}.
\end{DoxyReturn}
